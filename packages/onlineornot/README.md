`onlineornot` is a CLI for monitoring your uptime checks on [OnlineOrNot](https://onlineornot.com/).

## Quick Start

```
npm i -g onlineornot
onlineornot login
export ONLINEORNOT_API_TOKEN=api-token-goes-here
onlineornot checks list
```
